# tucaole-react

吐槽了，一个专注于吐槽的网站
react web

# Usage

Step 1. `git clone https://github.com/xuanheScript/tucaole-react.git`

Step 2.  
`cd tucaole-react`  
`npm install`  
`npm run start`
